In order to compile and run the main file full_cell_solver_new.c, you need to install Sundials 2.6.1. 
Download it from their website and install it using ccmake making sure that you keep the default options and enable lapack
It is necesssary to enable the default options else the Makefile will not be able to generate the right command to compile on your system
Run the Makefile in this directory.
This should produce an executable.
Run the executable to run the program
